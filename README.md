## Project Title

GUI Calculator Application

## Description

This project is a GUI calculator developed using Python Tkinter. It performs basic arithmetic operations through a graphical user interface.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Modulus (%)
* Exponent (^)
* Clear (C)

## How To Run

```bash
python calculator.py
```
